##### `Установка`

Убедиться в установке Python 3.8 и выше

- Для Linux-систем / Termux:

 `sudo apt update`
 
`git clone https://github.com/ernestlitvinenko/PythonSmsBomber.git`

`cd PythonSmsBomber`

`pip3 install -r requirements.txt`

`python3 bomber.py [Номер телефона для атаки]`
